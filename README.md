# showcase-admin-vue

Vue 3.js 空白项目模板

## 项目介绍

这是一个基于 Vue 3 + Vite 构建的空白项目模板，适合快速开始新的 Vue 3 项目开发。

## 技术栈

- **Vue 3** - 渐进式 JavaScript 框架
- **Vite** - 下一代前端构建工具
- **JavaScript ES6+** - 现代 JavaScript 语法

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发服务器

```bash
npm run dev
```

项目将在 http://localhost:3000 启动

### 构建生产版本

```bash
npm run build
```

### 预览生产构建

```bash
npm run preview
```

## 项目结构

```
showcase-admin-vue/
├── src/
│   ├── App.vue          # 根组件
│   ├── main.js          # 应用入口
│   └── style.css        # 全局样式
├── index.html           # HTML 模板
├── package.json         # 项目配置
├── vite.config.js       # Vite 配置
└── README.md           # 项目说明
```

## 开发说明

- 使用 Vue 3 Composition API
- 支持热重载开发
- 现代化的构建工具链
- 响应式设计支持
